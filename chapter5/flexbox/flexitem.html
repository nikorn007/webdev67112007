Flex Item Properties

เรียนรู้และทดลองการกำหนดค่า flex-grow, flex-shrink, flex-basis และ flex สำหรับ Flex Item

Flexbox และ Item Properties

Flexbox เป็นโมเดลการจัดเลย์เอาต์ใน CSS ที่ช่วยให้สามารถจัดเรียงและกระจายพื้นที่ระหว่าง Item ใน Container ได้อย่างมีประสิทธิภาพ

Flex Container จะมี Property สำหรับควบคุมการแสดงผลในระดับ Container ในขณะที่ Flex Item จะมี Property สำหรับควบคุมแต่ละ Item โดยเฉพาะ

Property สำคัญสำหรับ Flex Item ได้แก่:

  • flex-grow - กำหนดอัตราการขยายตัวของ Item เมื่อมีพื้นที่ว่าง
  • flex-shrink - กำหนดอัตราการหดตัวของ Item เมื่อพื้นที่ไม่เพียงพอ
  • flex-basis - กำหนดขนาดเริ่มต้นของ Item ก่อนที่จะขยายหรือหดตัว
  • flex - Shorthand property สำหรับกำหนดค่าทั้งสามพร้อมกัน

flex-grow

คำอธิบาย

flex-grow กำหนดอัตราส่วนการขยายตัวของ Item เมื่อมีพื้นที่ว่างใน Container ค่าเริ่มต้นคือ 0 ซึ่งหมายความว่า Item จะไม่ขยายตัว

ถ้า Item ทั้งหมดมีค่า flex-grow เป็น 1 พื้นที่ว่างทั้งหมดจะถูกแบ่งให้กับ Item อย่างเท่าๆ กัน ถ้า Item หนึ่งมีค่าเป็น 2 และอีก Item มีค่าเป็น 1 Item ที่มีค่าเป็น 2 จะได้รับพื้นที่ว่างมากกว่าอีก Item หนึ่งเท่าตัว

.item1 { flex-grow: 1; }
.item2 { flex-grow: 2; }
.item3 { flex-grow: 1; }
ตัวอย่างการทำงานของ flex-grow พื้นที่ว่างทั้งหมด: 500px

flex-shrink

คำอธิบาย

flex-shrink กำหนดอัตราส่วนการหดตัวของ Item เมื่อพื้นที่ใน Container ไม่เพียงพอ ค่าเริ่มต้นคือ 1 ซึ่งหมายความว่า Item จะหดตัวเมื่อจำเป็น

ถ้า Item หนึ่งมีค่า flex-shrink เป็น 0 มันจะไม่หดตัวเลย ทำให้ Item อื่นอาจต้องหดตัวมากขึ้นเพื่อให้พอดีกับ Container

.item1 { flex-shrink: 1; }
.item2 { flex-shrink: 0; } <-- จะไม่หดตัว
.item3 { flex-shrink: 2; } <-- จะหดตัวมากเป็น 2 เท่าของ Item 1
ตัวอย่างการทำงานของ flex-shrink พื้นที่ Container: 400px (Item รวมกันกว้าง 500px)

flex-basis

คำอธิบาย

flex-basis กำหนดขนาดเริ่มต้นของ Item ก่อนที่จะขยายหรือหดตัวตามค่า flex-grow และ flex-shrink

flex-basis สามารถกำหนดได้ด้วยหน่วยต่างๆ เช่น px, %, em, rem หรือใช้ค่า auto (ค่าเริ่มต้น) ซึ่งจะใช้ขนาดตาม content ของ Item

.item1 { flex-basis: 200px; }
.item2 { flex-basis: 30%; }
.item3 { flex-basis: auto; } <-- ขนาดตาม content

flex (Shorthand Property)

คำอธิบาย

flex เป็น shorthand property ที่รวม flex-grow, flex-shrink และ flex-basis ไว้ใน property เดียว

รูปแบบ: flex: flex-grow flex-shrink flex-basis;

ตัวอย่าง:

flex: 1; /* เท่ากับ flex: 1 1 0; */
flex: 0 0 200px; /* ไม่ขยาย ไม่หดตัว และมีขนาดเริ่มต้น 200px */
flex: 2 1 auto; /* ขยาย 2 เท่า, หดตัวปกติ, ขนาดตาม content */

สรุป Flex Item Properties

Property ที่สำคัญสำหรับการควบคุม Flex Item แต่ละอัน

flex-grow

ควบคุมการขยายตัวของ Item เมื่อมีพื้นที่ว่างใน Container ค่าเริ่มต้นคือ 0 (ไม่ขยายตัว)

ใช้งานเมื่อ: ต้องการให้ Item ขยายตัวเพื่อใช้พื้นที่ว่างทั้งหมด

flex-shrink

ควบคุมการหดตัวของ Item เมื่อพื้นที่ไม่เพียงพอ ค่าเริ่มต้นคือ 1 (หดตัวได้)

ใช้งานเมื่อ: ต้องการควบคุมว่า Item ใดควรหดตัวมากหรือน้อยเมื่อ Container แคบ

flex-basis

กำหนดขนาดเริ่มต้นของ Item ก่อนการขยายหรือหดตัว ค่าเริ่มต้นคือ auto (ตาม content)

ใช้งานเมื่อ: ต้องการกำหนดขนาดพื้นฐานของ Item ก่อนคำนวณพื้นที่ที่เหลือ

flex (shorthand)

รวมค่าทั้งสามไว้ใน property เดียว รูปแบบ: flex: grow shrink basis;

ตัวอย่าง: flex: 1 0 200px; (ขยายได้, ไม่หดตัว, ขนาดเริ่มต้น 200px)

ข้อแนะนำการใช้งาน

  • ใช้ flex-grow สำหรับการแบ่งพื้นที่ว่างอย่างเป็นสัดส่วน
  • ใช้ flex-shrink: 0 สำหรับ Item ที่ไม่ควรหดตัว (เช่น รูปภาพหรือ component บางประเภท)
  • ใช้ flex-basis ร่วมกับ flex-grow เมื่อต้องการกำหนดขนาดเริ่มต้นก่อนขยาย
  • ใช้ flex shorthand สำหรับความสะดวกรวดเร็ว
